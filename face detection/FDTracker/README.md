# FDTracker in MindSpore

## Introduction
MindSpore is a new generation of full-scenario AI computing framework launched by Huawei in August 2019 and released On March 28, 2020.

FDTrack surpasses the previous state-of-the-art (SOTA) methods and has good anti-interference and real-time performance. Our proposed modules have good portability and can be applied in other one-shot trackers to achieve performance improvement.
